About This Project

This website was built as the Milestone #1 deliverable for the IT 3203 - Intro to Web Development Class Project at Kennesaw State University. The project involves creating an informational website on a web technology topic across three milestones throughout the semester.

Project Milestones

  1. Milestone 1 - Basic website with HTML & CSS (complete)
  2. Milestone 2 - JavaScript-powered quiz application (this milestone)
  3. Milestone 3 - Responsive design for mobile devices

Topic: HTTP Protocol

The topic chosen for this project is the Hypertext Transfer Protocol (HTTP). The research explores how HTTP works, its history and evolution from HTTP/0.9 through HTTP/3, how HTTPS adds security, and how HTTP continues to power the modern web across APIs, cloud platforms, and mobile applications.

Technology Used

  • HTML5 - page structure and content
  • CSS3 - styling, layout (CSS Grid and Flexbox), and hover effects
  • JavaScript (ES6) - interactive quiz with form validation and dynamic grading
  • GitHub Pages - site hosting at austingrly.github.io
