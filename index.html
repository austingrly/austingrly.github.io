Introduction

When using the internet, you are always relying on the Hypertext Transfer Protocol (HTTP). It is present in every webpage you load, every image you view, and every form you submit. Browsers, servers, mobile apps, and cloud platforms all use this protocol to communicate. HTTP is the foundation of web communication and will continue to evolve as the web grows in size and complexity every day.

This site explores how HTTP works, how it has evolved from HTTP/0.9 to HTTP/3, the role of HTTPS and security, and how the protocol powers the modern web.

HTTP powers every server connection on the modern web.
